Computers

Hanszen offers free black and white printing for all Hanszenites. We have two printers; one in the Weenie Loft and another in the Laundry Room. (Simply provide your own paper.)

Contact me to get involved!
Devin Taylor • dtaylor@rice.edu

Access Instructions:

Weenie Loft Printer

Windows 7:

      1. Go to the Control Panel, and select Devices and Printers.
      2. Click on Add a Printer, and select the "Add a Network Printer" button.
      3. Click on "The printer that I want isn't listed".
      4. Select "Add a printer using a TCP/IP address or hostname" and click Next.
      5. In the hostname or IP address, type 10.129.26.5 and click Next.
      6. After the driver installs, you're good to go. Rename the printer if you want, and make sure not to share the printer.

Windows Vista:

      1. The printer is a HP LaserJet P2055dn PCL6 and the driver should be available. If not, go to the HP website and download the driver.
      2. Run the program you just downloaded, and extract the files to someplace you'll remember (such as the desktop).
      3. Go to the Control Panel, and select Devices and Printers.
      4. Click on Add a Printer, and select the "Add a Network Printer" button.
      5. Click on "The printer that I want isn't listed".
      6. Select "Add a printer using a TCP/IP address or hostname" and click Next.
      7. In the hostname or IP address type 10.129.26.5 and click Next.
      8. When prompted to select the driver, select "Have Disk", and if the driver is not found navigate to where you downloaded the file and select the correct file.
      9. After the driver installs, you're good to go. Rename the printer if you want, and make sure not to share the printer.

Windows XP:

      1. The printer is a HP LaserJet P2055dn PCL6 and the driver should be available. If not, go to the HP website and download the driver.
      2. Go to the Control Panel, and select Printers.
      3. Click on Add a Printer, and select the "Local Printer" button, and make sure that the Plug-and-Play checkbox is unchecked. Click Next.
      4. Click "Create a New Port", and in the drop-down box, click "Standard TCP/IP port". Click Next.
      5. In the new dialog box in the address field, type 10.129.26.5. Click Next.
      6. When prompted to select the driver, select "Have Disk", and if the driver is not found navigate to where you downloaded the file and select the correct file.
      7. After the driver installs, you're good to go. Rename the printer if you want, and make sure not to share the printer.

Mac OS X:

      1. The printer is a HP LaserJet P2055dn PCL6 and the driver should be available. If not, go to the HP website and download the driver.
      2. Run the file you downloaded and reboot your computer if the driver installation requests it.
      3. Click on System Preferences in the dock, and then click on "Print & Fax".
      4. In Printing, click the "+" button to add a new printer.
      5. Click on IP Printer at the top of the window. Select "Internet Printing Protocol - IPP" from the Protocol drop-down menu. Put 10.129.26.5 in the address field.
      6. At this point, make sure that the Name and "Print Using" fields at the bottom show that the printer is a HP LaserJet P2055dn series printer. Click Add.
      7. Once the driver is configured, you'll be finished.

Laundry Room Printer

Windows 7:

      1. Go to the Control Panel, and select Devices and Printers.
      2. Click on Add a Printer, and select the "Add a Network Printer" button.
      3. Click on "The printer that I want isn't listed".
      4. Select "Add a printer using a TCP/IP address or hostname" and click Next.
      5. In the hostname or IP address, type hanszennewsection.rice.edu and click Next.
      6. After the driver installs, you're good to go. Rename the printer if you want, and make sure not to share the printer.

Windows Vista:

      1. Download the printer drivers from the Brother website.
      2. Run the program you just downloaded, and extract the files to someplace you'll remember (such as the desktop).
      3. Go to the Control Panel, and select Devices and Printers.
      4. Click on Add a Printer, and select the "Add a Network Printer" button.
      5. Click on "The printer that I want isn't listed".
      6. Select "Add a printer using a TCP/IP address or hostname" and click Next.
      7. In the hostname or IP address, enter hanszennewsection.rice.edu and click Next.
      8. When prompted to select the driver, select "Have Disk", and navigate to the folder where you extracted the files. Select BROHL08A.INF and click OK. If you're prompted for the type of Brother printer, select the Brother HL-5370DW series.
      9. After the driver installs, you're good to go. Rename the printer if you want, and make sure not to share the printer.
